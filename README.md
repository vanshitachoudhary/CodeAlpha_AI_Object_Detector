# 🧠 Working Principle

The application follows the following AI pipeline:

1. Upload an image, video, or start webcam.
2. YOLOv8 processes every frame.
3. AI detects multiple objects.
4. ByteTrack assigns tracking IDs.
5. Bounding boxes are drawn.
6. Confidence score is calculated.
7. Detection analytics are displayed in real-time.
